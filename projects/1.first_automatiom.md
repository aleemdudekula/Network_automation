# 🚀 Network Automation Project – Python & Netmiko (GNS3 Lab)
I recently completed a hands-on network automation project using Python (Netmiko library) in a GNS3-based Cisco lab environment.

# 🔧 Project Overview
Built an automation script to securely connect to Cisco IOS devices via SSH and perform operational tasks programmatically.
##📌 Key Implementations:
Established SSH connectivity to Cisco routers using Netmiko
Automated execution of network CLI commands:
show ip interface brief
show ip route
Captured and processed real-time device outputs
Exported results into structured text files for logging and analysis

Basic network operations monitoring
File handling and output management
